Introduction

JavaScript is a versatile programming language used for web development, enabling dynamic and interactive content on websites.

It can be used for both client-side and server-side scripting, making it a core technology of the web alongside HTML and CSS.

  1. Download a modern browser like Chrome or Firefox.
  2. Set up a simple text editor or an IDE like Visual Studio Code.
  3. Create an HTML file and link a JavaScript file.
  4. Write a simple script using the <script> tag.
  5. Run the HTML file in your browser to see the results.
Java Entry

Despite the name similarity, JavaScript and Java are distinct languages. JavaScript does not have a 'main' method like Java does.

However, JavaScript code execution starts from the top of the script or from an event-driven trigger.

                
                    // JavaScript entry point
                    console.log("Hello, World!");

This simple script logs "Hello, World!" to the browser console, serving as a basic entry point.

JavaScript's event-driven nature means it often waits for user interaction to execute specific code blocks.

Printing to the console

One of the most common ways to debug JavaScript code is by printing output to the console using console.log().

The console can be accessed in most browsers by right-clicking the page, selecting "Inspect," and navigating to the "Console" tab.

                
                    console.log("This is a log message");
                    console.warn("This is a warning");
                    console.error("This is an error");

Each of these methods outputs messages of different severity levels, helping in effective debugging.

Other console methods include console.table() for tabular data and console.time() for performance tracking.

Declare a function

Functions in JavaScript allow you to encapsulate reusable logic. They are defined using the function keyword.

                
                    function greet(name) {
                        return "Hello, " + name + "!";
                    }

                    console.log(greet("Alice"));

This function takes a name argument and returns a greeting message.

JavaScript also supports arrow functions for more concise syntax:

                
                    const add = (a, b) => a + b;
                    console.log(add(5, 3));
Object Oriented Programming

JavaScript is a prototype-based, object-oriented language. Objects can be created using constructors or class syntax.

                
                    class Person {
                        constructor(name, age) {
                            this.name = name;
                            this.age = age;
                        }

                        introduce() {
                            console.log(`Hi, I'm ${this.name} and I'm ${this.age} years old.`);
                        }
                    }

                    const john = new Person("John", 30);
                    john.introduce();

Classes provide a clearer and more familiar way to work with object-oriented programming compared to older prototype-based methods.

Inheritance can be achieved using the extends keyword, allowing one class to inherit properties and methods from another.
